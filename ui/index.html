Welcome to MultiCIDR Shield! 🛡️

This plugin protects your network by blocking incoming connections from known malicious IP addresses.

How to use:

  • Find a reliable blocklist URL (must be plain text, like .txt or .csv).
  • Paste the URL into the input field above and click Add & Fetch.
  • The plugin will automatically download the list and start blocking traffic!
  • Lists are updated automatically every 6 hours.

Tip: The Data-Shield IPv4 Blocklist is a great starting point.
